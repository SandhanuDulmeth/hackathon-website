Elements fade in and slide up when they enter the viewport

This is an example of smooth scroll animations using the Intersection Observer API. Each element has a subtle slide-up motion (20-30px) combined with a fade-in effect. The animations trigger when elements are 20% visible in the viewport, creating a polished and professional user experience.

Performance

Uses CSS transforms and opacity for smooth, GPU-accelerated animations that maintain 60fps.

Accessibility

Respects user preferences with prefers-reduced-motion media query for better accessibility.

Modern API

Built with Intersection Observer API for efficient viewport detection without scroll listeners.

Customizable

Easy to customize timing, duration, and easing functions to match your design needs.

Cross-Browser

Works reliably across all modern browsers including Chrome, Firefox, Safari, and Edge.

Lightweight

No dependencies required. Pure vanilla JavaScript and CSS for minimal overhead.
